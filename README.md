# Pinterest-Data-Pipeline
Pinterest crunches billions of datapoints every day to decide how to provide more values to their users. 

![UML Diagram for Project](https://user-images.githubusercontent.com/89411656/167471689-7d694f4a-d7fc-4e94-bac4-302f57ffec54.jpg)

In this project, the data pipeline which Pinterest uses will be implemented. 

From pipelining data taken from a Pinterest API, to monitoring the process with tools such as Apache Airflow, Prometheus and Grafana. 

For this project, the Lambda architecture is used to take advantage of both batch and stream-processing. 

Data from the API is pipelined into Kafka, from which the procedures diverge into the batch and real-time streaming processes.
